## [1.0.0] - 2026-03-04

### Added
- Initial open source release
- `evidence.index` operator with governed field support
- `synth.json` operator with attempt state machine
- `enterprise_support_v1` policy preset (6 rules)
- `razi build` — deterministic compilation
- `razi validate` — spec and schema validation
- `razi run` — execution with full governance enforcement
- `razi replay` — offline deterministic replay
- Reference escalation workflow in examples/
- Three example input payloads (sla_breach, borderline, policy_violation)
